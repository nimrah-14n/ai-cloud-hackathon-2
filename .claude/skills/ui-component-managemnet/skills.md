# UI Component Management Skill

## Instructions

1. Use TypeScript + TSX for all components.
2. Use Tailwind CSS for styling; avoid inline styles.
3. Components must be modular, reusable, and responsive.
4. Include props and TypeScript typing for safety.
5. Components should integrate seamlessly with API responses.
6. Build components for:
   - Navbar
   - Task List
   - Task Item
   - Buttons (Add, Edit, Delete, Complete)
   - Modals & Forms
7. Ensure accessibility (ARIA attributes, semantic HTML).
8. Include responsive layouts for mobile, tablet, and desktop.
