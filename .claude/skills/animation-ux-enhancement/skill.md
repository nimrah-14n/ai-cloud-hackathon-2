# Animation & UX Enhancement Skill

## Instructions

1. Use React + TypeScript (TSX) for all components.
2. Use Tailwind CSS and optional Framer Motion for animations.
3. Implement animations for:
   - Fade-ins and slide-ins
   - Button hover and click effects
   - Task completion transitions
   - Modal open/close transitions
4. Keep animations subtle, fast, and professional.
5. Ensure animations do not block functionality or API calls.
6. Maintain responsive behavior for mobile and desktop.
7. Document examples of each animation type in `examples/`.
8. Provide reusable TSX templates in `templates/`.
