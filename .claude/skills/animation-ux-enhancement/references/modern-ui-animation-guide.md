# Modern UI Animation Guide

1. **Principles**
   - Subtle and purposeful animations
   - Feedback for user actions
   - Avoid overwhelming users

2. **Performance**
   - Use CSS transitions or Tailwind animations
   - Avoid heavy JS animations
   - Animate only necessary elements

3. **UX Patterns**
   - Fade-ins for new content
   - Hover/scale for interactive elements
   - Slide-ins for modals or sidebars
   - Strike-through + opacity for completed tasks

4. **Accessibility**
   - Avoid animations causing motion sickness
   - Provide visual cues along with animation
   - Ensure keyboard navigation works

5. **Integration**
   - Combine animations with API data updates
   - React + TSX for dynamic content
   - Tailwind classes or Framer Motion for advanced patterns
