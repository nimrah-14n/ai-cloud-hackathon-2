# Frontend Engineering Core Skill

## Purpose
This skill defines professional-grade frontend engineering practices for building scalable, accessible, and high-performance user interfaces using React, TypeScript (TSX), and modern UI patterns.

It ensures frontend behavior is predictable, spec-aligned, and production-ready.

## Scope
- Component architecture (atomic & feature-based)
- State handling (loading, error, empty, success)
- UX behavior consistency
- Accessibility & performance compliance
- Clean integration with backend APIs and specs

## Principles
- Spec-driven UI behavior
- Type-safe components (TypeScript)
- Reusable and composable components
- Clear UI state boundaries
- Accessibility-first design

## Used With
- Spec-Driven Development
- CRUD Core Skill
- API Management Skill
- Animation & UX Skill
